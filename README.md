# UH Sugar Land Student Center Website

This project is a front-end website for the University of Houston Sugar Land Student Center. It includes multiple pages, interactive features, and visual design aligned with UH branding.

---

## ✅ Pages Overview

### **homePage.html**

* Intro to the Student Center
* Navigation menu
* Hero section with image/video placeholder
* Overview text

### **About.html**

* Mission summary
* Facilities & services description
* Visual placeholders

### **RoomBooking.html**

* Room reservation form (5 rooms)
* Field validation (front-end)
* Confirmation placeholder

### **FAQPage.html**

* Collapsible FAQ sections (min. 5)
* Uses JavaScript for interactivity

### **Contact.html**

* Contact form
* Campus map placeholder

---

## ✅ Visual Design

* **Primary colors:** Red, White
* **Secondary colors:** Black, Light Gray
* **Font:** Sans-serif
* Tone: Clean, professional, welcoming

---

## ✅ Features

* Text-based navigation menu
* CSS styling throughout for consistent appearance
* Multimedia placeholders (image/video)
* FAQ accordion component
* Form validation on booking form
* Hover interaction
* Responsive-friendly layout

---

## ✅ Getting Started

1. Clone or download repo
2. Open `homePage.html` in a browser to start
3. Edit content/media within corresponding folders
